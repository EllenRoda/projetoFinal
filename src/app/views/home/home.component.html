O presente projeto (Zero Mega X) de estágio tem como objetivo o desenvolvimento de um sistema para uma loja de gamer, visando oferecer uma solução eficiente e prática para o gerenciamento de produtos, clientes e vendas no segmento gamer. O sistema proposto busca atender tanto a organização interna da loja quanto proporcionar uma melhor experiência para os clientes, tornando os processos mais ágeis e seguros. A proposta do sistema consiste em criar uma plataforma onde seja possível cadastrar, editar, consultar e excluir informações de produtos, como jogos, consoles, acessórios e itens colecionáveis, além de gerenciar dados de clientes, seus respectivos contatos e endereços. Para embasar o desenvolvimento deste projeto, são utilizados conceitos de programação orientada a objetos, modelagem de banco de dados e princípios de desenvolvimento web, além de práticas de boas regras de negócio aplicadas ao contexto comercial. O projeto contempla também um cronograma de execução com etapas bem definidas, além da definição dos recursos necessários, tanto de hardware quanto de software, que viabilizam o desenvolvimento e a implementação do sistema de gestão da loja de gamer.
